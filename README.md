# h2database-debs
Build Debian Package for H2 Database Engine

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- liblucene3-java
- libosgi-compendium-java
- libosgi-core-java
- libservlet2.5-java
- libslf4j-java
- libjts-java

**Tested**

- Debian jessie
